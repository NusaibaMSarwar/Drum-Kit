An interactive drum kit that lets you play drum sounds using your keyboard or mouse clicks.

## How to Play

1. Open `index.html` in your browser
2. **Click** on any drum button with your mouse, OR
3. **Press** the corresponding keys on your keyboard (W, A, S, D, J, K, L)
4. Each key plays a different drum sound!

## Key Mapping

- **W** - Tom 1
- **A** - Tom 2
- **S** - Tom 3
- **D** - Tom 4
- **J** - Snare
- **K** - Crash Cymbal
- **L** - Kick Bass

## Features

- 7 different drum sounds
- Click or keyboard controls
- Button animation on press
- Responsive design

## Technologies Used

- HTML
- CSS
- JavaScript
